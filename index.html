ShareOut

Cервис для передачи ссылок и файлов путём генерации коротких кодов доступа

Актуальность

Этот веб-сервис актуален в современном мире, где обмен информацией и контентом играет важную роль. Он позволяет пользователям генерировать открытые коды для передачи ссылок и текста без необходимости создавать аккаунты или раскрывать личные данные. Это удобно для различных сфер, таких как образование и бизнес, и обеспечивает безопасный обмен данными с минимальным риском злоупотребления.

К примеру:
В любых образовательных учреждениях возникает ситуация, когда нужно поделиться ссылкой на публичный компьютер без авторизации в каких-либо сайтах/мессенджерах. Также данная ситуация возникает повсеместно, у на работе и конференциях, если выступающий забыл флешку и т.п.

Структура веб страниц сайта

  1. Главная страница с описанием проекта

    • Первый экран с кратким описанием и кнопкой опробовать
      • Заголовок h1 с названием сервиса
      • Параграф с описанием работы
      • Кнопка переносящая на страницу с функционалом
    • Секция из двух колонок со сравнением возможностей с регистрацией и без
      • Заголовок колонки 1
      • Параграф о функционале без авторизации
      • Заголовок колонки 2
      • Параграф о функционале с авторизацией
      • Кнопка переносящая на страницу с регистрацией
    • Секция FAQ
      • Компонент Аккордеон каждый элемент отдельный вопрос
        • Заголовок с вопросом
        • Параграф с ответом
        • Опционально: необходимая ссылка

  2. Страница с экспресс функционалом (для роли гостя или роль пользователя)

    • Виджет публикации и получения контента (переключение через segmented button)
      • Форма публикации
        • Заголовок поля ввода ссылки
        • Поле ввода текста/ссылки
        • Слайдер длительности публикации
        • Кнопка генерации кода
        • Появляющийся блок с кодом
          • Содержимое блока
          • Кнопка скопировать код
      • Форма получения
        • Заголовок поля ввода кода
        • Поле ввода кода
        • Кнопка получения ссылки
        • Появляющийся блок с текстом
          • Содержимое блока
          • Кнопка скопировать текст

  3. Страница личного кабинета пользователя

    • Виджет публикации и получения контента (аналогично экспресс странице)
    • Виджет список текстового контента для публикации для каждого элемента списка:
      • Предпросмотр текста
      • Дата добавления
      • Кнопка подставки данных в форму
      • Кнопка публикации в 1 клик (заменяется на заглушку «Опубликовано»)

      • В конце списка: ссылка для перехода на страницу со всеми элементами
    • Виджет список файлов для публикации для каждого элемента списка:
      • Предпросмотр имени файла
      • Дата добавления
      • Кнопка подставки данных в форму
      • Кнопка публикации в 1 клик (заменяется на заглушку «Опубликовано»)

      • В конце списка: ссылка для перехода на страницу со всеми файлами

  4. Страница с полным списком файлов

    • Список – таблица со всеми файлами
      • Кастомное имя для файла (что находится в предпросмотре)
      • Название файла
      • Дата добавления файла
      • Расширение файла
      • Размер файла
      • Кнопка загрузить
      • Кнопка редактировать
      • Кнопка публикации в 1 клик (заменяется на заглушку «Опубликовано»)

      • Заголовок поля ввода
      • Поле ввода кастомного имени для файла
      • Имя файла, который редактируется
      • Кнопка сохранить

      • Содержимое
      • Дата добавления файла
      • Размер в символах
      • Кнопка копировать
      • Кнопка редактировать
      • Кнопка публикации в 1 клик (заменяется на заглушку «Опубликовано»)

      • Заголовок поля ввода
      • Поле ввода кастомного имени для ссылки

      • Поле ввода пароля
